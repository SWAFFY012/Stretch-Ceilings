# LANDOS - натяжные потолки

Статический лендинг для услуги натяжных потолков LANDOS.

## Что внутри

- `index.html` - разметка сайта.
- `styles.css` - адаптивные стили.
- `script.js` - калькуляторы, форма, меню и анимации.
- `assets/images/` - оптимизированные изображения для сайта.
- `PICTURES/` - исходные изображения и референс макета.

## Запуск

Откройте `index.html` в браузере. Сборка и сервер не требуются.
